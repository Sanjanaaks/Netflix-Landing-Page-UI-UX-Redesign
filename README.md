# Netflix-Landing-Page-UI-UX-Redesign
A complete redesign of the Netflix landing page created as part of the CODTECH Internship – Task 1. This project applies modern UI/UX principles to transform the original landing page into a cleaner, premium, and more cinematic experience.

## Project Objective
To improve usability, readability, visual hierarchy, and overall user experience while maintaining Netflix’s branding.

## Tools Used
- Figma
- Adobe Photoshop
- Canva
- Python (ReportLab, python-docx)

## Project Structure
```
Netflix-Landing-Page-Redesign/
├── Report/

├── Presentation/
├── Images/
└── README.md
```

## Deliverables
- Reports (PDF & DOCX)
- Portfolio Case Study (PDF)
- Presentation (PPTX)
- Before & After Images

## Final Outcome
A cleaner, cinematic UI with improved readability, spacing, CTA visibility, and visual hierarchy.
